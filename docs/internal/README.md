the Oxcer development process. These files are not required to use or contribute to Oxcer;
they are preserved here for historical context and for developers diving deep into
specific subsystems.

| VIEW_HIERARCHY_MEMORY_ANALYSIS.md | SwiftUI view-hierarchy memory analysis |
